## 映画予約プロジェクト ##

## 概要
SpringBootとJPAを使って実装した、学習用プロジェクトです。

## 機能
・会員登録、ログイン<br>
・映画リスト表示<br>
・予約管理（予約確認、キャンセル）<br>
・映画の登録<br>
・映画のスケジュール管理

## 技術スタック
・SpringBoot<br>
・Spring Web<br>
・Thymeleaf<br>
・Lombok<br>
・JPA(Hibernate)<br>
・PostgreSQL<br>
・QueryDSL

## パッケージ構成
・config「JPAQueryFactory設定用パッケージ」<br>
・controller「画面連携」<br>
・domain「エンティティ定義」<br>
・dto「データ転送オブジェクト」<br>
・repository「DBアクセス処理」<br>
・service「ビジネスロジック処理」
